# MAIL CLIENT

## Dependencies

In order to run this project, you need to install the following dependencies:


### Node JS and npm

On mac you can simply run:

```bash
brew install node.js
```
#### OR
[Install Node.js][]
[Install Node.js]: https://nodejs.org/en/download/
### Installing tools and dependencies


```bash
npm install
```


### Run proyect

```bash
npm run dev
```
